# 2D游戏开发技能

## 精灵系统 (Sprite Systems)

| 组件 | 用途 |
|------|------|
| **Atlas（图集）** | 合并纹理，减少绘制调用 |
| **Animation（动画）** | 帧序列 |
| **Pivot（轴心）** | 旋转/缩放原点 |
| **Layering（层级）** | Z轴顺序控制 |

### 动画原则
- 帧率：通常8-24 FPS
- 挤压与拉伸（冲击效果）
- 动作前的预备动作
- 动作后的跟随延续

## 瓦片地图设计 (Tilemap Design)

### 瓦片考虑因素

| 因素 | 建议 |
|------|------|
| **尺寸** | 16x16, 32x32, 64x64 |
| **自动平铺** | 用于地形 |
| **碰撞** | 简化形状 |

### 图层结构

| 图层 | 内容 |
|------|------|
| Background | 非交互背景 |
| Terrain | 可行走地面 |
| Props | 交互对象 |
| Foreground | 视差叠加 |

## 2D物理系统

### 碰撞形状

| 形状 | 适用场景 |
|------|----------|
| Box（矩形） | 矩形物体 |
| Circle（圆形） | 球类、圆润物体 |
| Capsule（胶囊） | 角色 |
| Polygon（多边形） | 复杂形状 |

## 摄像机系统

| 类型 | 用途 |
|------|------|
| **Follow（跟随）** | 追踪玩家 |
| **Look-ahead（预判）** | 预测移动方向 |
| **Multi-target（多目标）** | 双人游戏 |
| **Room-based（房间式）** | 类银河战士游戏 |

## 游戏类型模式

### 平台跳跃类 (Platformer)
- 土狼时间（Coyote time）：离开边缘后的宽限
- 跳跃缓冲
- 可变跳跃高度

### 俯视角 (Top-down)
- 8方向或自由移动
- 瞄准或自动瞄准
- 是否旋转角色

## 反模式（避坑指南）

| ❌ 不要 | ✅ 正确做法 |
|--------|-----------|
| 分离纹理 | 使用图集 |
| 复杂碰撞形状 | 简化碰撞 |
| 抖动摄像机 | 平滑跟随 |
| 像素完美+物理混用 | 选择一种方案 |

> **记住：** 2D游戏的核心是清晰度。每一个像素都应该传达信息。
